Available Commands

Automod Commands

/automod enable

Enable the automod feature to automatically manage inappropriate content.

/automod disable

Disable the automod feature if you no longer want automated management.

/automod setchannel

Specify the channel where the automod will send alerts for moderation actions.

/addfilter

Add keywords or phrases to the filter list to prevent unwanted content.

/removefilter

Remove keywords or phrases from the filter list to allow previously blocked content.

/viewfilter

Display the current list of active filters in the automod settings.

/whitelist user

Add a user to the whitelist to allow them to bypass automod restrictions.

/whitelist channel

Add a channel to the whitelist so that it is exempt from automod actions.

/whitelist role

Add a role to the whitelist, allowing all members of that role to bypass automod.

/unwhitelist user

Remove a user from the whitelist, restoring their automod restrictions.

/unwhitelist channel

Remove a channel from the whitelist, enabling automod actions in that channel.

/unwhitelist role

Remove a role from the whitelist, reapplying automod restrictions to its members.

Server log Commands

/server-logs enable

Activate server logs to start tracking and recording important server events.

/server-logs disable

Deactivate server logs to stop tracking and recording events in the server.

/server-logs setchannel

Specify the channel where server logs will be posted.

Moderation Commands

/ban

Remove a member from the server and prevent them from rejoining.

/kick

Remove a member from the server, allowing them to rejoin later.

/warn

Issue a warning to a member.

/warnings

View the list of warnings issued to a specific member.

/clearwarn

Remove all warnings for a specific member.

Other Commands

/help

Displays a list of all available commands and their descriptions.

/setlanguage

Change the bot's language to customize the experience for users.
